# 🚆 HimaRailConnect – Train Booking Web Application

HimaRailConnect is a simple and user-friendly Train Booking Web Application that allows users to search for trains and simulate the ticket booking process. This project was developed to demonstrate basic web development skills and to provide a clean interface for managing train travel information.

## 🌐 Live Demo
https://himarailconnect.netlify.app/

## 📌 Features
- Search trains between locations
- Train booking simulation
- Simple and clean user interface
- Easy navigation between pages
- Responsive layout for better user experience

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

## 🚀 How to Run the Project
1. Download or clone the repository.
2. Open the project folder.
3. Double-click `index.html`.
4. The application will open in your web browser.

## 🎯 Purpose of the Project
This project was created as a practice project to improve front-end development skills and understand how a basic train booking system works.
